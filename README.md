Projeto desenvolvido como desafio do curso DEVstart. Uma landing page de turismo apresentando Colmar, uma cidade no nordeste da França.

## Funcionalidades

- Banner com imagem de fundo e texto em overlay
- Navegação interna entre seções via âncoras
- Seção de apresentação da cidade com destaques
- Seção de apresentação da equipe
- Seção de dicas de viagem com cards
- Footer com links de navegação e disclaimer

## Tecnologias

- HTML5
- CSS3
- Google Fonts

## Como rodar

1. Clone ou baixe os arquivos do projeto
2. Adicione as imagens referenciadas no CSS e os placeholders de foto da equipe
3. Abra o `index.html` no navegador

## Estrutura

## Idioma

O conteúdo da página foi escrito em inglês como parte do processo de aprendizado e para demonstrar proficiência no idioma, aproveitando o projeto como oportunidade de praticar além do desenvolvimento web.

## Seções

- **Why Colmar?** — Apresentação da cidade com lista de destaques práticos
- **Meet Us** — Apresentação dos membros da equipe com foto e citação
- **Tips** — Cards com dicas de viagem de quem já esteve lá
